#xqjs
is a simple JavaScript console for Firefox that:

- executes JS
  (like [Execute JS](http://code.google.com/p/executejs/))
  within specified window
  (chosen in [DOMi](https://developer.mozilla.org/en/DOM_Inspector) style).
- can preprocess code with:
  - Macro written in JS
  - [CoffeeScript](http://jashkenas.github.com/coffee-script/)
